# 🌐 Personal Portfolio Website

A modern and interactive full-stack developer portfolio built to showcase my skills, projects, and experience. This portfolio reflects my passion for creating clean, responsive, and visually engaging web applications.

## 🚀 Features

- 🏠 **Homepage** – Engaging landing section with smooth animations and modern UI
- 👤 **About Page** – Detailed introduction, tech stack, and professional overview
- ⚠️ **Error Page** – Custom 404 page for better user experience
- 🎨 Smooth scrolling and animations using GSAP & Lenis
- ✨ Interactive UI components integrated from ReactBits
- 📩 Contact section with EmailJS integration
- 📱 Fully responsive design across devices

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Animations:** GSAP, Framer Motion, Lenis
- **3D & Visuals:** Three.js, OGL
- **Routing:** React Router DOM
- **Icons:** React Icons, Lucide React
- **Email Service:** EmailJS
